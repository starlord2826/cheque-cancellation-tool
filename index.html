How to use:

  1. Click "Capture from Camera" to take a photo with your device camera
  2. Or click "Upload Image" to select an existing image file
  3. Add "CANCELLED" text using either:
    • Option 1: Upload an image of "CANCELLED" written on blank paper - the tool will extract only the text
    • Option 2: Click "Add Digital Cancelled Text" to type and customize digital text
  4. Optionally upload a signature image
  5. Drag the uploaded elements onto the cheque to position them manually
  6. Click the rotate button (‚Üª) on any element to rotate it 90 degrees
  7. Drag the resize handle (green circle) at the bottom-right corner to resize elements
  8. Download the processed image

All processing is done locally in your browser. No data is sent to any server.
